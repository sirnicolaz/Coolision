Lista degli attributi dei prodotti. 
Type indica il tipo dell'attributo; values indica i possibili valori che può avere l'attributo (se è di tipo dropdown, altrimenti non ha senso); Req indica se è obbligatorio o no; code indica l'attribute code usato da magento per utilizzare l'attributo (una volta inserito in fase di creazione dell'attributo non può più cambiare); label indica il label usato da magento (una volta inserito in fase di creazione dell'attributo non può più cambiare); label ita indica il label usato nel frontend italiano (cioè nel sito italiano), label eng indica il label usato nel frontend inglese.

  type: text
  scope: global
  unique value: yes
  apply to: all
  req: yes
  code: sku
  label: SKU
  label ita: SKU
